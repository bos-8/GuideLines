1. Download zip file from: [More download option](https://www.apachefriends.org/pl/download.html)
   1. Chose xampp-portable-windows-x64-XXXXXXXXXXX.zip
2. Unzip folder to `C:\`
3. Add php to system PATH: `C:\xampp\php`
   1. Check in powershell:
```powershell
php -v
```
